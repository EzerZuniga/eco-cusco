# eco-cusco
Sistema de Gestión de Limpieza Pública para la ciudad del Cusco, desarrollado con Java y Spring Boot. Permite registrar, atender y monitorear reportes ciudadanos sobre acumulación de basura, puntos críticos, alcantarillas dañadas y mantenimiento de áreas verdes. Incluye API REST, base de datos estructurada y arquitectura en capas
